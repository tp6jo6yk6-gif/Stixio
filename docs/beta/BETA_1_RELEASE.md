# Stixio Beta 1 Release Notes

## Release name

Stixio Beta 1 — Quick Project Test

## What changed

- Rebranded app to Stixio.
- Refactored `index.html` into a small app shell.
- Moved app UI into `src/ui/stixio-app.js`.
- Added core architecture modules under `src/core`.
- Added Workspace core architecture under `src/core/workspace`.
- Redesigned UI into a product-style sticker workspace.
- Added Quick Project mode.
- Added Free mode one PNG export.
- Added Starter demo ZIP export for up to 8 stickers.

## Main test URL

## Primary beta task

Import a 4x4 AI-generated sticker sheet, apply layout, select one sticker, refine it, export one PNG, then try Starter demo ZIP export.

## Known limitations

- No real login yet.
- No real Google Drive yet.
- No manual crop handles yet.
- No brush repair yet.
- Multi-platform output rules are not fully active yet.

## Pass condition

The beta passes if a tester can complete the Quick Project flow without a blank page or fatal error.
